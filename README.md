# TypeScript Project Lab

A structured workspace documenting my hands-on journey learning **TypeScript** through small programs, data modeling exercises, and mini projects.

Instead of only studying theory, this repository focuses on **learning by building**, gradually moving from core TypeScript fundamentals to real backend-style patterns.

The goal is to understand how TypeScript improves **code reliability, scalability, and maintainability** when building modern JavaScript applications.

---

## Learning Approach

The learning protocol used in this repository follows three stages:

**Stage 1 вЂ” TypeScript Fundamentals**

Understanding the core syntax and type system:

* Type annotations
* Primitive types (`string`, `number`, `boolean`)
* Arrays and objects
* Function typing
* Type inference
* Union types
* Optional properties

---

**Stage 2 вЂ” Data Modeling**

Learning how real applications define structured data:

* Type aliases
* Interfaces
* Nested object structures
* Typed collections
* Modeling domain entities such as users, orders, and products

---

**Stage 3 вЂ” Practical TypeScript Usage**

Applying TypeScript concepts to small programs and preparing for backend development:

* Typed business logic
* Functional patterns
* Object immutability
* Preparing for TypeScript usage in APIs and backend services

---

## Running the Code

Install dependencies:

```
npm install
```

Compile TypeScript:

```
npx tsc
```

Run compiled JavaScript:

```
node dist/<file>.js
```
